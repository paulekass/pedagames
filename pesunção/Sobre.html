Sobre o Projeto

Bem-vindo ao Pedagames, um projeto desenvolvido pelos alunos da faculdade CCI com o objetivo de criar uma ferramenta educacional para a área de pedagogia. Nosso projeto foi concebido para atender tanto os alunos quanto os professores, oferecendo uma experiência interativa e educativa através de um site funcional e intuitivo.

Objetivo

O principal objetivo do Pedagames é fornecer um ambiente de aprendizado divertido e engajador para as crianças. Através de um minigame especialmente desenvolvido, os alunos podem aprender enquanto se divertem.

Para os Alunos

Os alunos têm acesso a um minigame interativo que foi projetado para estimular o aprendizado de forma divertida. A interface amigável e os desafios do jogo ajudam a manter as crianças engajadas e motivadas a aprender.

Para os Professores

Os professores têm uma página dedicada onde podem acompanhar o progresso de seus alunos. Através de um sistema de pontuação (score), os educadores podem visualizar o desenvolvimento de cada aluno, identificar áreas de melhoria e adaptar suas estratégias de ensino conforme necessário. Este recurso visa facilitar o acompanhamento do desempenho dos alunos e apoiar os professores em seu papel educacional.

Agradecimentos

Agradecemos seu interesse e esperamos que o Pedagames seja uma ferramenta valiosa tanto para os alunos quanto para os professores.
